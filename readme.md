## Usage

[`!include`](https://plantuml.com/preprocessing#393335a6fd28a804) the theme file:

```puml
@startuml

!include https://raw.githubusercontent.com/brunofrancoucu/plantuml-dark-contrast-theme/main/theme.puml

Bob->Alice : hello

@enduml
```
